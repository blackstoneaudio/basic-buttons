Basic Buttons

Why reproduce similar code every time you need buttons for a new project? Start with basic-buttons.less then customize to your needs. Over 200 icons are already included and the file is structured to make customization quick and easy.

Customizing

Adding new colors is really easy with the provided mixin

&.newcolor	{ .color-btn(colorOfBtn, txtColor); }

Changing the provided colors is pain free too, just change the {{LESS}} variables at the top of the file.

@btn-color-red		: #c0392b;
@btn-color-green	: #27ae60;
@btn-color-sage		: #c6e59a;
@btn-color-blue		: #3498db;
...
